核心判断
「真正难的从来不是让 Agent 跑起来，是让它在生产里不胡说、不超时、不被人绕过。」

我把每个 Agent 项目都当成一个分布式系统来设计：工具调用要有超时和降级、上下文要有边界、失败要有可重入路径。在 6 个业务域的实践中，这套工程纪律比任何 Prompt 技巧都更可靠。
