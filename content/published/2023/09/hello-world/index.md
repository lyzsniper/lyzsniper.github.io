# Hello World

欢迎来到 Hexo！这是你的第一篇文章。查看 [Hexo 文档](https://hexo.io/docs/) 了解更多。如果使用过程中遇到任何问题，可以在 [troubleshooting](https://hexo.io/docs/troubleshooting.html) 或在 [GitHub](https://github.com/hexojs/hexo/issues) 上提问。

## Quick Start

### Create a new post

```bash
$ hexo new "My New Post"
```

更多信息：[Writing](https://hexo.io/docs/writing.html)

### Run server

```bash
$ hexo server
```

更多信息：[Server](https://hexo.io/docs/server.html)

### Generate static files

```bash
$ hexo generate
```

更多信息：[Generating](https://hexo.io/docs/generating.html)

### Deploy to remote sites

```bash
$ hexo deploy
```

更多信息：[Deployment](https://hexo.io/docs/one-command-deployment.html)
